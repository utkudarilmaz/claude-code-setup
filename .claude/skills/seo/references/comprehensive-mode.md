# Comprehensive Mode: `/seo all`

Perform a planned, full-project SEO/GEO/AIO optimization covering every aspect of web content.

**Do not skip any aspect. Continue until ALL areas are optimized.**

## Execution Flow

1. **Discover all web content** - Identify HTML files, templates, sitemaps, robots.txt, and structured data
2. **Create TodoWrite plan** - One todo item per optimization aspect
3. **Process sequentially** - Complete each aspect before moving to the next
4. **Mark progress** - Update todos as each aspect completes

## Optimization Aspects to Process

Process each area one-by-one:

| Aspect | What to Optimize |
|--------|-----------------|
| Title & meta descriptions | Length (50-60 / 150-160 chars), keywords, uniqueness per page |
| Heading hierarchy | Single H1, logical H2-H6 nesting, keyword inclusion |
| Structured data (JSON-LD) | Schema.org types, completeness, validity, correct properties |
| Open Graph & social meta | og:title, og:description, og:image, og:url, Twitter Cards |
| Image optimization | Alt text, lazy loading, width/height attributes, format hints |
| Internal linking | Descriptive anchor text, link depth, orphan page detection |
| URL & canonicalization | Clean slugs, canonical tags, hreflang for multilingual content |
| Sitemap & robots.txt | Completeness, accuracy, crawl directives, sitemap index |
| Page speed hints | Preload, preconnect, dns-prefetch, async/defer on scripts |
| Semantic HTML & accessibility | HTML5 elements (article, section, nav, main), ARIA landmarks |
| GEO: Entity & citability | Definitions, factual density, FAQ/How-to schemas, passage optimization |
| AIO: Machine readability | JSON-LD coverage, context density, document outline, structured tables |

## Example TodoWrite Plan

When `/seo all` is invoked, create todos such as:

- [ ] Optimize title tags and meta descriptions across all pages
- [ ] Fix heading hierarchy (single H1, logical nesting)
- [ ] Add or improve JSON-LD structured data (Schema.org)
- [ ] Add or improve Open Graph and Twitter Card meta tags
- [ ] Optimize image alt text and add lazy loading
- [ ] Improve internal linking and anchor text
- [ ] Add canonical URLs and verify URL structure
- [ ] Review and update sitemap.xml and robots.txt
- [ ] Add page speed resource hints (preload, preconnect)
- [ ] Improve semantic HTML structure and accessibility
- [ ] Optimize for GEO: entity clarity, FAQ schemas, passage structure
- [ ] Optimize for AIO: machine readability, context density, document outline

Dispatch the seo-optimizer agent for each aspect sequentially, marking each complete as it finishes.
